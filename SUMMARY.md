# Rogmor - A Next.js MMORPG Project

## Project Overview
Rogmor is a browser-based MMORPG (Massively Multiplayer Online Role-Playing Game) built with Next.js and TypeScript. The project aims to create a minimal yet engaging RPG experience with a focus on simplicity and modern web technologies.

## Key Features
- 2D graphics and tile-based maps
- Character-based combat system
- Custom "Slash" scripting language for skill programming
- Real-time combat encounters
- Character progression and attributes system
- Team-based combat mechanics

## Technical Stack
- **Frontend Framework**: Next.js 12.1.6
- **Language**: TypeScript
- **State Management**: Redux with Redux-Saga
- **Styling**: Tailwind CSS
- **Testing**: Jest, Cypress
- **Database**: MongoDB
- **API**: GraphQL with Apollo Server

## Core Systems

### Character Attributes
- **Body**: Physical strength and capabilities
- **Reflex**: Reaction time and quick thinking
- **Soul**: Spiritual harmony and wisdom
- **Aura**: Social presence and impact
- **Stamina**: Health and endurance
- **Focus**: Mental stability
- **Morale**: Social interaction energy

### Combat System
- Turn-based combat mechanics
- Team-based encounters
- Custom skill system using "Slash" scripting
- Auto-fight capabilities
- Speed-based action ordering

### Development Features
- Test-Driven Development (TDD) approach
- Comprehensive testing suite
- Modern development tools and practices
- Mobile-responsive design
- Progressive Web App (PWA) capabilities

## Getting Started
1. Clone the repository
2. Install dependencies: `yarn install`
3. Run development server: `yarn dev`
4. Run tests: `yarn test`
